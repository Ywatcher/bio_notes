# B Cell Mechanisms Rough Sources

Rough bibliography for papers mentioned while asking how the BCR / immunoglobulin rearrangement loop works biologically and algorithmically.

This bucket is not a final curated bibliography. It is a holding area for papers that may explain:

- V(D)J recombination constraints and feedback.
- RAG-created DNA breaks and DNA-damage checkpoints.
- Pre-BCR / BCR signaling and allelic exclusion.
- RAG transcription, FOXO1, PI3K/AKT, MAPK/ERK, and related signaling.
- Ig-locus targeting by RSSs, chromatin state, and 3D genome architecture.
- Memory of rearranged sequences, germinal center selection, somatic hypermutation, class switching, and memory B-cell output.

Use this before deciding which entries should stay in the main topic `references.bib`.
