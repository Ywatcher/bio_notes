# Source Notes

Created on 2026-07-16 from the discussion about whether BCR-generation loops or B-cell biology can be engineered for proteins with purposes beyond ordinary immune response.

Questions this bucket came from:

- Can engineered B cells produce useful proteins when the desired sequence is unknown or only partly known?
- How does engineering B cells compare with display systems, directed evolution, CAR-T-like strategies, and direct gene therapy?
- Which works modify the BCR / Ig locus process itself, rather than merely using natural antibodies?
- What safety mechanisms are being explored for engineered B cells?

Selection status:

- Rough / mentioned, not carefully ranked.
- Some papers engineer BCR specificity directly; others engineer B-lineage cells as protein-secreting platforms.
- Majors et al. 2012 is included because it uses somatic hypermutation machinery for directed evolution of a non-antibody protein.
