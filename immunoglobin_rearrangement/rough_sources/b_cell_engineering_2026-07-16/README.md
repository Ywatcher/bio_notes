# B Cell Engineering Rough Sources

Rough bibliography for papers mentioned while asking whether the BCR / B-cell system can be engineered, not only studied or leveraged.

This bucket is not a final curated bibliography. It is a holding area for papers about:

- Editing endogenous immunoglobulin loci.
- Redirecting B-cell receptor specificity.
- Engineered B cells or plasma cells as durable therapeutic protein producers.
- Safety controls for engineered B cells.
- Using B-cell somatic hypermutation machinery for directed evolution.

Use this before deciding which entries should stay in the main topic `references.bib`.
